Os arquivos H5 armazenam dados em HDF (Hierarchical Data Format). Esses arquivos são usados para armazenar dados numéricos, gráficos ou de texto. 
Os srquivos H5 são comument chamados de bibliotecas de dados. justamente por armazenar muitas informações que são utilizadas nos mais variados tipos de projetos.

O formato H5 é usado por muitos setores, incluindo aeronáutica ou ramos de ciências, como astronomia, medicina, finanças, física.

Esses arquivos também são usados para fins acadêmicos e educacionais. A lista completa de softwares compatíveis pode ser encontrada no site do Grupo HDF.
